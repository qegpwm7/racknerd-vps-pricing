3. **SSH in immediately.** `ssh root@your.server.ip` with the emailed password. First thing: change the root password with `passwd`, then disable root login over SSH and add your own key. Basic hygiene, takes two minutes.

4. **Update and harden.** Run `apt update && apt upgrade -y` (or `dnf` equivalent on Alma/Rocky). Install `ufw`, allow only 22 (or your custom SSH port), 80, 443. Enable unattended-upgrades if you don't want to babysit patches.

6. **Reinstall OS anytime from the panel.** SolusVM lets you swap distros with one click — Debian to Arch, Ubuntu to Alma, whatever. Useful when you accidentally rm something you shouldn't have.

7. **Set up monitoring.** Even a free UptimeRobot ping every five minutes catches silent failures. Don't skip this. The first time your box goes down at 4 a.m. and you find out from a user email instead of an alert is the last time you skip it.
